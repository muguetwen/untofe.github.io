【Dockerfile】构建一个镜像

发表于2023-05-24|更新于2024-09-25|LinuxDocker
|阅读量:

基本流程

1.构建一个空文件夹并进入

mkdir newdir
cd newdir

2.创建一个Dockerfile文件

touch Dockerfile

3.编写Dockerfile文件。

vim Dockerfile

使用该命令进入Dockerfile文件

FROM ubuntu
RUN mkdir dir1

或者

echo 'FROM ubuntu'>Dockerfile
echo 'RUN mkdir dir1'>>Dockerfile

Dockerfile 常用指令:

FROM 以什么为基础
MAINTATINER 镜像维护者姓名或邮箱
WORKDIR 指定工作目录
RUN 运行
CMD  运行容器时执行
ARG  指定一些参数
EXPOSE 声明端口
VOLUME 指定容器挂载的目录
USER 指定用户
ENV 设定容器环境变量
ADD 拷贝文件或目录
COPY 与ADD类似，但不具备自动下载或解压
ENTRYPOINT 执行shell命令

4.构建镜像

docker build -t  镜像名:tag .
